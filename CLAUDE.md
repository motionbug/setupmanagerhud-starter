## Project Overview

Setup Manager HUD is a real-time webhook dashboard for [Setup Manager](https://github.com/jamf/setup-manager). This is a **starter template** that wraps the `@motionbug/setupmanagerhud-core` npm package and deploys to Cloudflare Workers.

## Commands

```bash
npm run dev      # Start local development server (wrangler dev)
npm run sync     # Sync assets/migrations from the core package
npm run deploy   # Apply D1 migrations and deploy to Cloudflare Workers
npm run upgrade  # Update core package and sync new assets
```

Set the webhook token secret:
```bash
npx wrangler secret put WEBHOOK_TOKEN
```

## Architecture

This is a thin deployment wrapper around `@motionbug/setupmanagerhud-core`:

- **src/index.ts** - Entry point that re-exports the core app and DashboardRoom Durable Object
- **wrangler.toml** - Cloudflare Workers configuration with D1 database and Durable Objects bindings
- **public/** and **migrations/** - Synced from the npm package (gitignored, regenerated by `npm run sync`)

### Cloudflare Bindings

- `DB` - D1 database for event persistence
- `DASHBOARD_ROOM` - Durable Object for real-time WebSocket connections
- `WEBHOOK_TOKEN` - Secret for webhook authentication
- Optional: `CF_ACCESS_AUD` and `CF_ACCESS_TEAM_DOMAIN` for Cloudflare Access protection

### Customization

The entry point can be extended by wrapping `app.fetch` or adding custom routes before delegating to the core handler.
